## 🚀 Deployment

### Deploy to GitHub Pages

```bash
# 1. Clone the repository
git clone https://github.com/ishakeel98-ux/dropship-mastery-2026.git
cd dropship-mastery-2026

# 2. The site is ready — no build step required
# GitHub Pages serves index.html directly

# 3. Enable GitHub Pages in repository settings:
#    Settings → Pages → Source: Deploy from main branch
```

### Local Development

```bash
# Option 1: Python HTTP server
python -m http.server 8000

# Option 2: Node.js http-server
npx http-server -p 8000

# Then open: http://localhost:8000
```
